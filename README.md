# City - Smart

Plataforma de una ciudad inteligente desarrollada por BeeStudents utilizando Tecnologías para FrontEnd, UI/UX y acceso a alginas API's públicas.

## Características

- Login y Registro de Usurarios.
- Dashboard Responsivo.
- Integración con FIrebase (Pasos Evolutivos).
- Consumo de API´s Públicas (Clima, Noticias, Calidad del Aire)

## Tecnología Utilizada (Stack de Desarrollo)

- HTML5.
- CSS3.
- Bootstrap 5.
- Javascript.
- Firebase.
